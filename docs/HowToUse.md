#### 仕様

- SplatNet2仕様のフォーマット
- SplatNet3の生データのフォーマット

のどちらも取得できます. 使いやすい方をご利用ください.

> Salmon Stats+にアップロードする際にはSplatNet2のフォーマットに準拠する必要があります.

### 導入方法

```
yarn add @salmonstats3/av5ja
```
